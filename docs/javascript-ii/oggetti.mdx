## Intro

## Creazione

Per creare un oggetto in JavaScript, possiamo utilizzare la sintassi degli oggetti letterali, che ci consente di definire un oggetto con una serie di proprietÃ  e valori:

```jsx

  cognome: 'Rossi',
  eta: 30,
  professione: 'Ingegnere'
```

Possiamo accedere alle proprietÃ  dell'oggetto utilizzando la notazione punto o la notazione delle parentesi quadre:

```jsx

```

Possiamo anche aggiungere o modificare le proprietÃ  dell'oggetto in qualsiasi momento:

JavaScript ha anche una serie di oggetti predefiniti che possiamo utilizzare. Ad esempio, l'oggetto String ci consente di lavorare con le stringhe:

```jsx

console.log(nome.length); // 5
console.log(nome.toUpperCase()); // 'MARIO'
console.log(nome.charAt(0)); // 'M'
```
